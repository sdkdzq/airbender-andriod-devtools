Enable PerfHUD ES on the Nexus 7

Requirements

  • Nexus 7 device
  • Build number JDQ39 (Android 4.2.2 kernel 3.1.10-g05b777c)

Deployment Instructions

  1. Open the Nexus 7 distribution folder.
  2. Be sure that adb.exe is in the PATH environment, and that the device has already been rooted.
  3. Connect one device to the PC and run deploy.cmd.
