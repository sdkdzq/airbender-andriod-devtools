First Boot

The first time you start up your SHIELD device, you should see the Android welcome sequence (SHIELD portable welcome sequence shown below).

After booting, it is recommended that you do the following: 

  1. Enable the developer mode by going to Settings > About SHIELD, scroll to the bottom and tap Build Number several times. (See Unlocking Developer Mode for more info.) 
  2. Go to Settings > Developer Options and enable Stay awake in order to avoid the SHIELD device turning itself off after some period of inactivity.
  3. It is also recommended that you go to Settings > Display and change the Sleep time to 30 minutes.

The device should be displayed when you run adb devices from a command prompt. If it is not displayed, the Windows USB driver installation may have failed; in this case, refer to instructions under How to Install Drivers on the Install AndroidWorks page to troubleshoot.

You are now ready to use NVIDIA AndroidWorks tools. Please follow the instructions on how to use each product in the corresponding sections.
