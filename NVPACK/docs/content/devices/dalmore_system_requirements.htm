System Requirements

Software Requirements

The basic requirements for Tegra 4 development are:

  • A development PC running Windows
  • (Optional) Visual Studio 2010 Service Pack 1, including Visual C++ (for the Nsight Tegra Visual Studio plugin)

All other required software is installed by the TADP installer. This includes:

  • OS and OS Install Components
    • The USB driver for Dalmore’s “recovery” mode (the mode used to install OS images)
    • The Android OS image (Jellybean in the current installer)
    • The Android Tegra OS “flashing” (install) app, NvFlash.exe
    • A script that flashes the OS
  • Android Development Components
    • The ADB USB driver
    • The Android SDK and NDK
  • NVIDIA Development Tools
  • The Nsight Tegra Visual Studio plugin for developing Android applications in Visual Studio

Hardware Requirements

Dalmore devkit hardware supplied in the box includes the following:

  • Main devkit board
  • Debug/expansion board
  • Power adapter
IMPORTANT: Unlike most previous-generation Tegra development kits, the Dalmore uses an AC adapter that outputs 12V DC (previous kits normally used 15V DC). Verify that you are using a 12V adapter, and contact your NVIDIA Developer Relations manager if you do not have a 12V adapter.

Developer-supplied hardware includes:

  • Windows PC (XP, Vista or Win-7, x32 or x64) with an available USB port
  • USB A to USB-micro cable (may or may not be shipped with the development kit; this is not a custom cable. Standard USB A to USB micro cables should work)
