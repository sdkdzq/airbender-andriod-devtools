System Requirements

Software Requirements

The basic requirements for SHIELD development on a Windows PC environment are: 

  • SHIELD running the latest OS from Software Update OTAs
  • (Optional) Visual Studio 2010 SP1, 2012, or 2013 Professional (or higher) Editions, including Visual C++ (for Nsight Tegra, Visual Studio Edition) 

All other required software is installed via the NVIDIA AndroidWorks installer. This includes:

  • Android Development Components
    • ADB USB driver
    • Android SDK and NDK
    • Build-Tools and Platform-Tools
  • NVIDIA Developer Tools
    • Nsight Tegra, Visual Studio Edition
    • Tegra System Profiler
    • Tegra Graphics Debugger
  • Tegra Development Documentation and Guides
  • Android Code Samples

Hardware Requirements

SHIELD hardware supplied in the box includes the following:

  • An NVIDIA® SHIELD™ family device
    • SHIELD
    • SHIELD portable
    • SHIELD tablet
  • Power adapter and micro-USB cable
  • Starter's guide

Developer-supplied hardware includes:

  • Windows PC (XP, Vista or Windows 7 or Windows 8, x32 or x64) with an available USB port
  • USB-A to USB-micro cable (may or may not be shipped with the development kit; this is not a custom cable. Standard USB-A to USB-micro cables should work.)
