Tegra System Profiler 1.x User Guide

Overview

Tegra System Profiler performs profiling of your application and provides you with an interactive view of the results, intended to help you improve overall performance.

With Tegra System Profiler, you can:

  • Identify call paths that monopolize the CPU.
  • Identify individual functions that monopolize the CPU (across different call paths).
  • See how CPU usage and cache misses changed during the sampling period.
  • Identify how well your application’s CPU usage is balanced across multiple CPU cores (if present).

The Tegra System Profiler application runs on a host Windows or Linux computer (for example, a desktop or laptop), and launches or attaches to an application on a target Android device that meets specific compatibility requirements, and is connected to the computer via USB. Tegra System Profiler installs and uses a special daemon on the device to perform the sample-based profiling and communicate the results back to the main application.

This is a beta release of the Tegra System Profiler software. While the core functionality planned for the first release has been completed, you should expect to encounter some bugs and a general lack of polish throughout the application. This software has been provided to you as a preview, in hopes that both the customer and NVIDIA may benefit through providing developers with data to assist in optimizing their applications, while developers in return optionally provide feedback, suggest improvements, and report bugs.

Release notes are documented in the Known Issues section.
