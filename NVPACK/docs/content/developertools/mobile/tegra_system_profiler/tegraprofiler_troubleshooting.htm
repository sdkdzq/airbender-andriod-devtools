Troubleshooting

When reporting any bugs please include the build version number from the Help > About dialog, as well as the log files from the target device.


While this image may not be updated for every document revision, it may be viewed as a reasonable representation of what to expect.

Log Files

There is one log file on the target device: /data/data/com.nvidia.tegraprofiler.tools/files/data/quad_d.log.

To copy the log file from the target device to the host, run the following command in a console window (cmd.exe) on the host:

adb pull /data/data/com.nvidia.tegraprofiler.tools/files/data/quadd_d.log c:\quadd_d.log
