AndroidWorks 1R2

NVIDIA AndroidWorks reduces the complex process of configuring an Android development environment with a single mouse click, providing a full suite of native application development tools for Android, now available for all Android platforms. New to this release is a full version of IncrediBuild 7.0 for Visual Studio. NVIDIA has worked closely with IncrediBuild to accelerate build times when using IncrediBuild for Visual Studio in conjunction with the latest Nsight Tegra, Visual Studio Edition 3.1. In addition, this release includes updated developer tools (Tegra System Profiler, Nsight Tegra, Visual Studio Edition), installation UI and usability improvements, and the latest Android NDK, SDK, and Build/Platform Tools for Android development.
