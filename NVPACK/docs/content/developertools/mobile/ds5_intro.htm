How To: Analyze Application Performance with ARM DS-5

How to Start Profiling with DS-5 Streamline

To start profiling applications with ARM DS-5 you need to do the following steps:

  1. Install the latest DS-5 on your PC (to date, only the Windows version has been tested).
  2. Configure the target kernel to support DS-5, and build it.
  3. Build the DS-5 kernel module called Gator Driver (gator.ko) and the build gator daemon (gatord).
  4. Setup profiling.
