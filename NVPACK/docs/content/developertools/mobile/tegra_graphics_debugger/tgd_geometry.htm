Geometry

The Geometry window takes the state of the OpenGL machine and the parameters for the current draw call and shows the pre-transformed geometry.

On the top left of this window, the clone and lock buttons work the same on this screen as they do on the API Inspector.

At the bottom of the window, there is a drop-down menu where you can select if you want to view Wireframe, Point Cloud, or Shaded geometry.
