Getting Started

There are two major components in getting Tegra Graphics Debugger installed and configured for debugging. These are: 

  • OS installation
  • Tegra Graphics Debugger installation

These steps will be outlined in more detail below.

OS Installation

Most Android devices, such as the SHIELD tablet, will work with Tegra Graphics Debugger out of the box, and don't need to be pre-configured.

However, if you are using a beta developer kit, such as an Ardbeg devkit, you may need to configure the device before using Tegra Graphics Debugger. For information on how to do this, see the Ardbeg OS installation page.

Tegra Graphics Debugger Installation

You can install Tegra Graphics Debugger as part of the NVIDIA AndroidWorks package (see Getting Started with AndroidWorks), or install as a standalone application.

On Windows

Run the .msi file and follow the prompts to install.

On Mac

Load the DMG, and copy the app to your applications folder.

On Linux

Run the .run file.
