Current Target

The Current Target view is used to show the currently bound output targets. This can be useful because it focuses in on the bound output resources, rather than having to search for them in the Resources view.

Current Target will display thumbnails along the left pane for all currently bound color, depth, and stencil targets. You can then select one of these, to show a larger image which is displayed on the right.
