Resources View

The Resources window allows you to dynamically cast your buffer and visualize it in 3space. There are four pages available here: 

  1. All Resources
  2. All Resources (Table)
  3. Graphical
  4. Memory

Resources

The All Resources page allows you to view information about textures (input resources), render buffers (output resources), buffers (data which may have revisions), and default frame buffer drawables. The buttons at the top serve as a coarse filter to display different resource types. The filter box allows you to type characteristics to look for to further limit the displayed thumbnails.

All Resources (Details) 

The All Resources tab also has a control to specify different visualizations, including large/small thumbnails, and a details mode. The details mode takes the resource information, and puts it in a table format. As the columns are sortable, this can help you narrow down a specific resource when searching.

Graphical

The Graphical page allows you to view more in depth details about the resource.

Memory

The Memory page shows a raw memory dump of the resource, allowing you to cast it to various float and integer types, as well as a hexadecimal display capability.
