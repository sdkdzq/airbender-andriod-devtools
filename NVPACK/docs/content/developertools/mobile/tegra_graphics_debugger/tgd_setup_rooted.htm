Using the Debugger: Rooted

This guide will give you a walkthrough of using Tegra Graphics Debugger with a rooted device.

Launching Applications with Tegra Graphics Debugger: 

  1. Start the Tegra Graphics Debugger Host from the Start menu.
    The Connect to Application dialog will appear and default to your connected device.
    If you have multiple devices connected, you can choose the drop-down menu next to Target and select the device you want to use.


  2. The Tegra Graphics Debugger GUI supports launching applications with the interposer preloaded. This requires pushing the interposer .so file to the target system. Once you select your target device, the button in the upper right corner will show the current status of the interposer.


    If the button has an X, the interposer is not on the target device. Click the button to push the interposer to the target device.

    If you see an error in the log saying “failed to copy” and “Read-only file system,” open a DOS box and use the adb root and adb remount commands to allow the files to be uploaded.
  3. Choose the apk you are interested in debugging from the list of applications, and press the Launch button in the lower right hand on your device. Once the application has started you should see the main screen transition to the Performance Dashboard.

Attaching to Applications with Tegra Graphics Debugger

  1. If you have linked your application with the Tegra Graphics Debugger and launched it by tapping on the application icon on the device, select the tab that says Attach, and click the refresh button just above the Application Name list to see the application of interest.


  2. Applications that are enabled for Frame Debugging will appear in the list.
    1. You can choose Show All if you want to view all applications that are running on the target device; applications that cannot be debugged will appear with their names grayed out.
    2. To quickly locate a specific application by name, you can type the first few letters of its name, or its process ID in the Search field, and only matching applications will appear in the list.
  3. Select the application and click Attach in the lower right corner to start using the Tegra Graphics Debugger.
