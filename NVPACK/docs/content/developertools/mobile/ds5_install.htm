Install DS-5

  1. Download DS-5 from arm.com.
  2. It is recommended that the user review the DS-5 documentation prior to installation.
  3. Run the Windows installer file . For version 5.14, this is called DS500-BN-00003-r5p0-14rel0.zip.
  4. The license file is called license.dat.
    The license is for a 30-day trial period. It is auto-generated when you login to arm.com and download DS-5.
  5. Upon the first DS-5 run, enter the license number.
Note: Hereinafter, $DS-5 refers to your DS-5 installation directory.
