Nsight Tegra 3.1, Visual Studio Edition

Overview

NVIDIA Nsight Tegra is a full Android development environment that is integrated into Microsoft Visual Studio.

This user guide covers the functions of Nsight Tegra and outlines how you can use Nsight Tegra to debug your Android applications.

Getting Started

Installing Nsight Tegra with AndroidWorks

NVIDIA AndroidWorks is a simplified solution for Android developers, providing you with a single installer that manages the complex setup environment for you on Windows 7 (64-bit). Nsight Tegra is included with AndroidWorks, to help you debug your Android applications.

Development tools installed include the following: Android SDK, Android NDK, JDK, Eclipse, ADT, and Apache Ant.

After you download and install AndroidWorks from http://developer.nvidia.com/androidworks, you should have a fully-configured Android development environment ready for development, as well as the additional Tegra specific tools required to do in-depth debugging and analysis on Tegra devices.

If you are developing on a Tegra Development Kit, please download the latest OS Image and flash the device. This will ensure you have the most up-to-date system with the latest optimizations. (For more specific information, see Install AndroidWorks.) 

Installing Nsight Tegra without AndroidWorks

Nsight Tegra can also be run as a standalone application, without installing the entire AndroidWorks package. In order to do so, your machine needs to be running Visual Studio, and have the following environment variables set: 

  • ANDROID_HOME, ANDROID_NDK_ROOT
  • JAVA_HOME
  • JAVA_HOME\bin in path
  • ANT_HOME and/or GRADLE_HOME

Software Requirements

The following components are automatically installed with NVIDIA AndroidWorks. However, if you are running Nsight Tegra without AndroidWorks, you'll need to verify that each of the items below has also been installed:

.NET Framework v4.5 http://www.microsoft.com/download/en/details.aspx?id=17851
MS Visual Studio 2010 SP1, 2012, or 2013 Professional (or higher) Editions Available via MSDN subscription downloads.

Android SDK r24.2.0 or newer http://developer.android.com/sdk/index.html
Android NDK r9b — r10e http://developer.android.com/sdk/ndk/index.html
Latest JDK 1.7.0_71 or higher http://www.oracle.com/technetwork/java/javase/downloads/index.html
Apache ANT 1.8.2 or higher http://ant.apache.org/bindownload.cgi

First Use

After installing both AndroidWorks and Nsight Tegra, connect your supported Android device, open Visual Studio, and you will see that the NVIDIA Tegra Device Toolbar has been added.

You can now begin using Nsight Tegra to debug your Android applications.
