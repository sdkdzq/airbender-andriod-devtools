Using IncrediBuild with Nsight Tegra

Nsight Tegra works with IncrediBuild 5.5 (and newer) out-of-the-box. Setting the C/C++ > General > Maximum Number of Parallel Build Processes property is not advised, as this value is always hard-coded to a large enough number when building with IncrediBuild.

There is one extra property related to the IncrediBuild support, namely General > Force IncrediBuild Execution (Experimental). Setting this property to Yes will force the tool execution when building with IncrediBuild.

Changing this setting may result in increased building speed, but the tool output, reported errors, and warnings will be lost. This feature is experimental and should be used at your own risk.
