OpenCV Release Notes

New Features and Improvements

  1. Open-source improvements:
    • Support of Android up to 4.2, NDK r8c/r8d and SDK r21
    • Improved OpenCV Manager: less clicks on first use and no application restart required.
    • Added Android applications framework, reorganized samples, and updated tutorial.
    • Added MIPS architecture support.
    • Multiple fixes in OpenCV.mk and OpenCVConfig.cmake.
  2. Tegra 3 and 4 optimizations were added for the following: 
    • imgproc/Laplacian (kernel <= 5)
    • imgproc/Sobel (float point)
    • imgproc/matchTemplate(CV_TM_CCORR)
    • photo/fastNlMeansDenoising and photo/fastNlMeansDenoisingColored
    • imgproc/warpAffine and imgproc/warpPerspective have improved speed (up to 10x)
  3. Some functions have experienced minor speed-up or slow-down, due to moving from gcc-4.4.3 to gcc-4.7 (NDK r8d)
