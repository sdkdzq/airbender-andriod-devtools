OpenCV for Tegra Demo Application

An Android application called OpenCV_for_Tegra_Demo.apk demonstrates some examples of the optimization factors that can be achieved with OpenCV for Tegra versus OpenCV for Android on Tegra devices. This application is available in the <path to TADP directory>/OpenCV-2.4.8.2-Tegra-sdk/apk/ folder. The OpenCV_for_Tegra_Demo.apk application can also be freely downloaded from the Google Play store.

A sample screen shot of the OpenCV for Tegra Demo application is below.

Typical optimizations factors observed for the various modes of the demo application on the Tegra 4 device are listed below.
