System Requirements

The Tegra Android Natives Samples Pack is designed for use as a part of normal Android NDK application development. As a result, the development system onto which the samples pack is installed must be configured for Android NDK development.

Normally, it is installed in one of two ways:

Installed as Part of TADP

When installed as a part of TADP, the pack is immediately ready to use via the TADP-shipped Eclipse workspace.

Installed as Part of the Tegra Android Toolkit

If this pack was installed via the Tegra Android Toolkit, then it is assumed that the developer previously installed all of the prerequisites for Android NDK development, including:

  • The Android SDK including SDK versions 9 through 14
  • The Android NDK, version r7b or newer
  • Eclipse with CDT support
  • The Android Development Tool plugin for Eclipse (ADT)

Note that all of these items (and more) are automatically installed by TADP; installing the pack via a full TADP install is recommended.
