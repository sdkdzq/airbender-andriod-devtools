Release Notes

CodeWorks for Android 1R7

What’s New

  • Updated suite of Android NDK, SDK, Build/Platform Tools to latest available version. (Refer to Features for more details.) 
  • Nsight Tegra 3.5, Visual Studio Edition
    • Support for Android NDK64 r14
    • Support for API 24
    • Various performance improvements and bug fixes
  • Tegra Graphics Debugger 2.6
    • Android O support
    • Vulkan 1.0 support

      Note the following limitations with Vulkan support:

      1. Real Time GPU signals are not yet available.
      2. The Range Profiler is not supported.
      3. Dynamic shader editing is not yet available.

      These will be added in a future release.

    • Added OpenGL Pixel History
    • Range Profiler improvements
    • Various bug fixes and performance improvements
  • NVIDIA System Profiler 3.9
    • Various bug fixes and performance enhancements
  • PerfKit 4.5.1
  • IncrediBuild 9.2 for Visual Studio
  • NVIDIA GameWorks OpenGL Samples 20180130-github-snapshot
