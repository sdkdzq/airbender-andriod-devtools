NVIDIA CodeWorks for Android 1R7

NVIDIA CodeWorks for Android reduces the complex process of configuring an Android development environment with a single mouse click, providing a full suite of native application development tools for Android, now available for all Android platforms. New to this release are updated developer tools, the latest Android NDK, SDK, and Build/Platform Tools for Android development.

What's Included in CodeWorks for Android? 

NVIDIA CodeWorks for Android includes the following tools:

Android Development

The latest versions of: 

  • Android SDK
  • Android APIs
  • Android NDK
  • Google USB Driver
  • Android Support Library
  • Gradle
  • Apache Ant

NVIDIA Libraries and Tools

  • Tegra Graphics Debugger
  • Nsight Tegra, Visual Studio Edition
  • NVIDIA System Profiler
  • NVIDIA PerfKit
  • NVIDIA® SHIELD™ Android TV Support
  • NVIDIA® SHIELD™ Tablet Support

Third Party Middleware and Tools

  • IncrediBuild for Visual Studio

The pack provides sample applications, support libraries, and documentation on Android game development topics.

Where can I find it?

Download the latest version under NVIDIA GameWorks Download Center. Archived builds are available online at CodeWorks for Android Archives.
