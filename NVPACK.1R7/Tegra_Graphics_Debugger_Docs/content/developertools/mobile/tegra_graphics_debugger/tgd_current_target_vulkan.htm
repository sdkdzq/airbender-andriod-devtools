Current Target

The Current Target view is used to show the currently bound output targets. This can be useful because it focuses in on the bound output resources, rather than having to search for them in the Resources view.

Current Target will display thumbnails along the left pane for all currently bound color, depth, and stencil targets. You can then select one of these, to show a larger image which is displayed on the right.


Current Target color view


Current Target depth view

In order to see the buffer that you've selected on your device, you must have the HUD enabled. If you disable the HUD, the rendering on the target device will revert to color target 0.

Clicking on the eraser icon in the top left corner of this view will insert a clear call before the current action. The current target view will update to show the result, with the additional clear call.
